## Prompt contract

Each prompt requested polished hand-drawn anime game-sprite illustration in an
exact borderless 3 x 2 grid on a uniform `#00ff00` chroma-key field, with one
complete pose per cell, consistent identity/scale, generous padding, and no
opponent, scenery, text, panel, border, watermark, floor, shadow, or gore.

The actor sheets use rich cel shading and luminous high-gamma highlights, but
the runtime does not apply a blanket screenshot color grade. The chapter-502
scene instead carries a deterministic actor-only gamma/gain envelope around the
5150 ms contact frame.

The green backgrounds were removed locally with the installed imagegen helper
using border auto-key, soft matte, threshold 12/220, and despill. All keyed
sheets, pose atlases, and the wide 16:9 review composite were visually reviewed.

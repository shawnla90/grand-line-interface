Generated on 2026-07-18 with the built-in OpenAI image-generation tool. Every
sheet is an original fan-art interpretation created for the Grand Line
Interface internal prototype. No anime frame, manga panel, official texture,
franchise logo, or ripped game model was supplied or traced.

The approved East Blue Zoro or Smoker source sheet was used only as a local
style and identity-consistency reference. Outputs were generated on flat
chroma-key backgrounds, copied into this workspace, keyed locally, and visually
reviewed before atlas construction. Source sheets are production evidence;

## Prompt contract

Each prompt required a borderless 3×2 sheet, six named full-body states,
consistent early-story identity and costume, generous cell padding, a perfectly
flat removable key, and no text, logo, watermark, gore, environment, or copied
production frame. Character recognition at map scale is the visual acceptance
gate.

The Batch A+ ship prompt used the approved Ace sheet and Arabasta scene board
as rendering/readability references. It requested the same five-ship convoy in
six cells: approach, closing formation, Fire Fist impact, engulfed fleet, hull
breakup, and burning wreckage. The prompt explicitly excluded people, Marines,
naval insignia, official logos, water/sky backgrounds, translucent smoke, and
elements crossing cell boundaries.

The priority Batch B prompts produced six new actor sheets for Zoro/Mr. 1,
Sanji/Bon Clay, and Crocodile round one. Each prompt named the exact scene-useful
pose order and era-specific constraint. Sanji explicitly excludes Diable Jambe;
Luffy explicitly excludes the later water and blood countermeasures; Crocodile
explicitly requires visible human facial features in every sand state. This is
the visual replacement for the older rigid `arabasta-luffy-crocodile-sand-study`,
whose dark faceless blockout remains motion reference only.

The Arabasta-completion prompts added Nami/Miss Doublefinger and two separate
Crocodile rematch variants. Nami excludes Perfect Clima-Tact and Zeus. Round two
uses water only. The royal-tomb final uses restrained matte countermeasure
smears, poison-hook escalation, and upward Gum-Gum Storm, with no gore. Two
built-in image edits enforced runtime invariants: Luffy's actor sheet cannot
contain Crocodile, and final-round countermeasure marks cannot read graphically.
